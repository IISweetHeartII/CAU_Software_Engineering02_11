# 소프트웨어 설계 문서 (SDD)

## 1. 시스템 아키텍처
### 1.1 전체 구조
- MVC (Model-View-Controller) 패턴 적용
- 모듈화된 구조로 확장성 고려

### 1.2 주요 컴포넌트
- GameEngine: 게임 로직 관리
- Board: 게임판 상태 관리
- Main: 프로그램 진입점

## 2. 클래스 설계
### 2.1 GameEngine
- 게임 상태 관리
- 턴 진행 로직
- 승리 조건 검사

### 2.2 Board
- 게임판 초기화
- 말 이동 검증
- 게임판 상태 업데이트

## 3. 데이터 구조
### 3.1 게임판
- 8x8 2차원 배열
- 각 칸의 상태 표현

### 3.2 게임 상태
- 현재 플레이어
- 게임 진행 상태
- 승리 조건

## 4. 알고리즘
### 4.1 말 이동
- 이동 가능 위치 계산
- 충돌 검사
- 경로 검증

### 4.2 승리 조건
- 승리 조건 검사
- 게임 종료 처리
